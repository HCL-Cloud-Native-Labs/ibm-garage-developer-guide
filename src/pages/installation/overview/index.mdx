## Paths to Installation

The IBM Garage for Cloud Developer Tools have been configured so they can install into a number of
Kubernetes-based cluster technologies. The tools can be installed into multiple environments. 
Select the approach that best suites yours:
- If its a brand new project, use the *New* path
- If you have and existing IBM Cloud managed cluster, use the *Existing* approach
- If you have OpenShift 4 installed locally, use the *CodeReady Containers* path.
