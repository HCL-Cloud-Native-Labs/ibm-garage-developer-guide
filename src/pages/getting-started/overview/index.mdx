## Paths to Installation

the IBM Garage for Cloud Developer Tools have been configured so they can install into a number of
Kubernetes based cluster technologies. They can also be installed in different ways. Select the path
best suites your setup. If its a brand new project use the *New* path, if you have and existing IBM Cloud
managed cluster use the *Existing* approach. If you learning or needing to work remotely use the *CodeReady Containers* path.
